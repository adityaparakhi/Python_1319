List Comprehension and Set Comprehension
++++++++++++++++++++++++++++++++++++

Original List 1:  [1, 2]
Original List 2:  ['a', 'b']

Output using normal loop: [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]       
Original List 1:  [1, 2]
Original List 2:  ['a', 'b']

Output using list comprehension: [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]
Original List 1:  [1, 2]
Original List 2:  ['a', 'b']

Output using set comprehension: {(1, 'a'), (2, 'b'), (2, 'a'), (1, 'b')}
